
# Ecommerce 2024 

A full stack MERN ecommerce website


## Installation

Install pre-requisites such as

 mongodb

 redis

 windows subsystem for linux

 run "redis server" command on ubuntu terminal



- ### Frontend

  cd Ecommerce-Frontend

  npm i

  npm run dev


- ###  Backend

  cd Ecommerce-Backend

  npm i

  npm run dev

  
Add the .env file aswell
